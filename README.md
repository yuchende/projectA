  (2)后台程序-----------------------采用hibernate+servlet
  (3)后台数据库---------------------mysql

❥❥❥本电影小程序功能


（1）点赞功能，（当前台点击 点赞按钮，先在前台做加减，再发送更新点赞数量的请求到后端）
（2）浏览电影，（当点击某一个电影，跳转播放页面，采用用户缓存做页面信息传输）
